# Discord Bot written in C#

NuGet packages used:
- Discord.Net v2.1.1
- Victora v4.0.17
- MingweiSamuel.Camille v2.2.1
- TMDbLib v1.0.0
- JikanDotNet v.1.2.5
- Colorful.Console v1.2.9

Commands:
  League:
    @pm nudes elo <summonerName>
  Music:
    @pm nudes join
    @pm nudes move
    @pm nudes leave
    @pm nudes play
    @pm nudes pause
    @pm nudes resume
    @pm nudes stop
    @pm nudes skip
    @pm nudes volume or @pm nudes vol
    @pm nudes songinfo or @pm nudes current
    @pm nudes queue
    @pm nudes clearqueue or @pm nudes cq
  Discord:
    @pm nudes profile or @pm nudes profile <name>
    @pm nudes guild
    @pm nudes +rep <name>
    @pm nudes -rep <name>
  Extra:
    @pm nudes movie <name>
    @pm nudes anime <name>
    @pm nudes gif
